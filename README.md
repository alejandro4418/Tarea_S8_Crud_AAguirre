# Sistema de Registro Institucional - CRUD con Herencia y ArrayList en Java

## 📌 Descripción del Proyecto

Este proyecto es una aplicación de consola desarrollada en Java que permite gestionar el registro de personas pertenecientes a una comunidad académica. El sistema administra dos tipos de personas: **Estudiantes** y **Docentes**, aplicando los principios fundamentales de la Programación Orientada a Objetos (POO).

## 🎯 Objetivo de Aprendizaje

Aplicar conceptos de POO en Java mediante una aplicación funcional que integre:
- Herencia
- Encapsulamiento
- Sobrescritura de métodos
- ArrayList para almacenamiento de datos
- Operaciones CRUD
- Menú interactivo
- Manejo de excepciones con try-catch

## 🏗️ Estructura del Proyecto
src/
├── Persona.java # Clase padre
├── Estudiante.java # Subclase que hereda de Persona
├── Docente.java # Subclase que hereda de Persona
└── Main.java # Clase principal con menú y CRUD

## 📝 Resumen de Validaciones del Código

✅ El programa compila sin errores  
✅ El menú no se cierra si se ingresan letras  
✅ El CRUD funciona con estudiantes y docentes  
✅ El ArrayList almacena objetos de tipo Persona  
✅ El código está modularizado con métodos separados  
✅ Se aplican buenas prácticas de POO
